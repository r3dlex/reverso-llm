# Work Intake

Local fallback work-intake surface for `reverso`. Work items recorded here
must be reconciled with the hosted tracker (GitHub issues for
`r3dlex/reverso-llm`) before a final PR merge.

Each work item is a Markdown file with: title, BRD/PRD link (if any), acceptance
criteria, and a traceability node ID (`issue:reverso-root:<slug>`). Hosted-tracker-first
is the preferred path when authorized; this directory is the local fallback per
the init-ai-repo Govern & Plan phase.
